Description of the Challenge

ImageCLEFmed concept detection 2021 sub-task is part of the ImageCLEFmed Caption task that aims to research and propose methodologies for automatic image captioning, with the purpose to speed up the process of interpreting and summarizing the insights gained from medical image such as radiology images.

You can find more about ImageCLEF 2021 results, and the different research tasks that were available for people to participate here.

Description of the Content Based Image Retrieval System

Content based image retrieval (CBIR), also known as query by image content, consists in the application of computer vision techniques to solve problems that involve searching images in databases. As the name suggests, the search analyzes the contents of the image instead of focusing on the metadata, focusing on particular aspects of the image such as colors, shapes, textures or any other information that might be derived from it. This alleviates the problem of relying on annotation techniques and quality for images that might be hard to find annotated.

The following picture gives an overview of how CBIR works, and it is worth mentioning that within a CBIR system, there are two main components: Feature Extraction and Similarity Measure. Feature Extraction, when it comes to images, consists in applying techniques that are able to process images and extract meaningful features from images such as shapes, colors, textures, etc. Similarity Measure consists in performing a comparison of your query image and the images in the database, this comparison can be done by comparing the distance between the features. Once you have a set of most similar images, it comes to deciding what will be recommended to the user, it can be that you are trying to predict the type of flower that is on the query image.
